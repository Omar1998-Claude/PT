# 📈 Portfolio Tracker PWA

A personal stock portfolio tracker with goals, dividends, quarterly review and injection planner.

## 🚀 Deploy to GitHub Pages (5 minutes)

### Step 1 — Create a new GitHub repo

1. Go to [github.com](https://github.com) and log in
1. Click **“New”** (green button top left)
1. Name it: `portfolio-tracker`
1. Set to **Public**
1. Click **“Create repository”**

### Step 2 — Upload your files

1. On the new repo page, click **“uploading an existing file”**
1. Drag and drop ALL files from this folder:
- `index.html`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`
1. Click **“Commit changes”**

### Step 3 — Enable GitHub Pages

1. Go to your repo **Settings** tab
1. Scroll to **“Pages”** in the left sidebar
1. Under **Source**, select **“Deploy from a branch”**
1. Select branch: **main**, folder: **/ (root)**
1. Click **Save**

### Step 4 — Get your app URL

After ~2 minutes your app will be live at:
`https://YOUR-GITHUB-USERNAME.github.io/portfolio-tracker`

### Step 5 — Install on your phone

**iPhone:** Open the URL in Safari → Share → Add to Home Screen  
**Android:** Open in Chrome → 3-dot menu → Add to Home Screen

That’s it! Your portfolio tracker is now a real installable app. 🎉

## Features

- 📊 Dashboard with holdings & allocation
- 💉 Monthly injection planner (Rotation & Index modes)
- 💰 Dividend income tracker with projections
- 🎯 Annual goal milestones ($5K → $15K → $35K)
- 🔍 Quarterly portfolio health review
- 📈 Injection history with growth chart
- 🌙 Dark & Light mode
- 💾 Data saved locally on your device
- ✈️ Works offline
